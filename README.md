# Run tracker Web Application

C# .NET web app using razor pages designed to log the date and distance of runs. Implements CRUD operations using an SQLite database. 

## Demo

![Demo Animation](../demo/run_tracker_demo.gif)


## Features

* Display logged runs in table with following details:
    * Date
    * Distance (km)
* Create new run from form
* Update timed session from form
   * Provide confirmation
* Delete timed session from form
    * Provide confirmation of deletion

## Pre-requsites 

### Dependencies 

* .NET 10.0 installation

## Run Locally

Clone the project

```bash
  git clone git@github.com:cesca2/runtracker_webapp.git
```

Go to the project directory

```bash
  cd runtracker_webapp
```

Run the application

```bash
  dotnet run
```

Open provided link in browser
